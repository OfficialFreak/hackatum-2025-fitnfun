# FitNFun
<img style="width: 10rem;" src="https://github.com/user-attachments/assets/af2a6804-bcac-4d23-af40-2af5bbcc9e64">

FitNFun is our HackaTUM 2025 submission for the Logitech challenge. It is a plugin developed using the Logi Actions SDK to help users stay healthy during long and stressful workdays.
It adds three new commands:
<img width="339" height="323" alt="image" src="https://github.com/user-attachments/assets/dd39c6c5-fea3-4c98-a8c5-03307a8c45f8" />

which try to support the user by lowering stress, reminding them of the 20-20-20 rule for their eyes and doing yoga excercises.
One part of the Repository is the plugin, the other the python prototype of the yoga pose detection.
